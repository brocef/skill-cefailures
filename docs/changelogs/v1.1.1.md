# v1.1.1

## Fixed

- `scripts/broker_cli.py` is now committed with the executable bit set, so the symlink setup instructions work on fresh clones without a manual `chmod +x`.

## Changed

- Broker setup docs now default to symlinking into `~/.local/bin` (no `sudo` required) and document `/opt/homebrew/bin` / `/usr/local/bin` as alternatives.
- Removed the now-redundant `chmod +x` step from `skills/broker/docs/setup.md`.
