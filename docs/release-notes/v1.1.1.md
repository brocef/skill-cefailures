# v1.1.1

## Fixes

- Broker setup now works end-to-end from a fresh clone. `broker_cli.py` ships with the executable bit set, and the setup docs no longer instruct a `chmod +x` as a follow-up step.
- The setup instructions now default to installing the `broker` symlink into `~/.local/bin`, so they work without `sudo` on a typical macOS/Linux setup. `/opt/homebrew/bin` and `/usr/local/bin` are listed as alternatives.
