# v1.1.0

## New skill

- **broker** — reference for the broker CLI, covering subcommands, server/client modes, and setup. Replaces the prior MCP-based broker tooling.

- Broker rewritten around a Unix domain socket transport with a subcommand-based CLI (`broker_cli.py`). The former MCP broker and its install scripts have been removed; use the CLI subcommands instead.
