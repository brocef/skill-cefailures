## Install Knex.js Core Package (Node.js)

```bash
npm install knex --save
```

## Install Database Drivers for Knex.js (Node.js)

Choose the driver corresponding to your database.

```bash
npm install pg
$ npm install pg-native
$ npm install sqlite3
$ npm install better-sqlite3
$ npm install mysql
$ npm install mysql2
$ npm install oracledb
$ npm install tedious
```

## Install Knex.js Migration CLI Globally

```bash
npm install knex -g
```
